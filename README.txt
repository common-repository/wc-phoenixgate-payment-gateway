== Description ==

With PhoeniXGate, you can offer your merchants a complete omni-channel payments solution, instantly boosting customer reach through one single connection. Using a single consolidated payment gateway to offer customers a wide array of payment options is now an essential component in any merchant solution. The PhoeniXGate payment gateway enables physical and virtual payment devices to co-exist, including taking advantage of the unique, secure and fully managed Phoenix payment infrastructure.<br>
Note: The plugin now supports WordPress Network sites, aka WP Multisite.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the payment gateway settings including the API URL, username and password those you get from phoenixmanagednetworks
4. You can enable e-check payment gateway
